# wget
## 简介

* 用于下载文件等
* 常用选项
  * **-P**：表示下载到哪个目录
  * -r：表示递归下载
  * -k：表示将下载的网页里的链接修改为本地链接
  * **-c**：断点续传
  * -nd：递归下载时不创建一层一层的目录，把所有的文件下载到当前目录
  * -np：表示不下载旁站连接
  * -i：后面跟一个文件，文件内指明要下载的URL

## 案例

1.下载 repo 到 /etc/yum.repos.d/ 目录下

```
$ wget http://mirrors.163.com/.help/CentOS7-Base-163.repo -P /etc/yum.repos.d/
```
