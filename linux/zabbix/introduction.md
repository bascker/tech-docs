# Zabbix
## 简介
**zabbix**是企业中应用比较多的**C语言**开发的基于 Web 界面的**开源分布式监控系统**，具有 2 个组成部分：
* zabbix-server：默认端口 10051
* zabbix-agentd：默认端口 10050

zabbix 具有如下特点：
* 多语言支持
* 开源，意味着免费
* 自动发现：可以自动发现同网段内的 agent 进程、网络设备、文件系统等
* 分布式
* 支持web界面
* 支持邮件、电话、短信、微信等通知功能
* 支持监控数据实时绘图
* 提供 API
* 支持权限控制
* 高性能、低内存消耗、易移植
